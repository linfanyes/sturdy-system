# 园丁工作台 · 微信小程序 · 系统说明书

> 适用对象：一线教师 / 班主任
> 版本：小程序端（uni-app），配套 NestJS 后端 + 腾讯云数据库 MySQL 8.0
> 本文档说明**产品能做什么、怎么用、数据存在哪里**，不涉及部署（部署见《微信小程序前后端完整部署与发布流程》）。

---

## 一、产品简介

**园丁工作台**是面向教师的班级管理一站式工具。小程序版把桌面版的核心能力搬到了手机上：班级管理、学生档案、考试与成绩、座位与分组、AI 备课生成、18 款课堂小游戏，以及语文/数学/英语专项练习、文字办公等共 **53 个页面、44 类工具**，全部可在微信里随手使用。

**设计原则**
- 所有密钥（微信、AI）只在后端配置，手机端不存储，安全合规。
- 数据按教师隔离：你只看到自己班级的数据。
- 小游戏、口算、倒计时等工具**无需联网**即可使用；班级数据、AI 生成需联网并经后端。

## 二、系统架构（用户视角）

```
┌──────────────┐     联网（HTTPS）     ┌──────────────────┐     联网     ┌─────────────────┐
│  微信小程序   │ ───────────────────► │  后端服务（云托管） │ ─────────► │  云数据库 MySQL  │
│  （你的手机） │ ◄─────────────────── │  NestJS          │ ◄───────── │  （你的数据）    │
└──────────────┘                      └──────────────────┘            └─────────────────┘
```

- **你（老师）**：在微信里打开小程序，输入/查看班级数据，使用各类工具。
- **后端服务**：负责登录鉴权、保存数据、调用 AI 生成内容。由学校/管理员部署在「微信云托管」。
- **数据库**：你的班级、学生、成绩等数据存放在这里，按老师分库隔离。

> 你不需要懂上面这些也能用。只要管理员把后端和数据库部署好，并告诉你小程序的名字/二维码，你微信里打开即用。

## 三、登录与账户

1. 首次打开小程序，点击「微信登录」（授权获取微信昵称/头像，仅用于标识账号）。
2. 系统用你的微信身份签发登录态（JWT），后续操作自动带上。
3. 登录态长期有效；若提示「请重新登录」，重新点一次微信登录即可。
4. **多设备**：同一微信账号在另一台手机登录，数据互通（数据存在云端，不在本机）。

## 五、核心教学管理

### 5.1 班级管理
- 新建/编辑班级（年级、学科、班主任等），学生归属到班级。
- 班级列表默认按学号升序展示学生。

### 5.2 学生管理
- 学生档案：姓名、学号、所在学校/班级、座次号、职务、家长联系方式。
- 支持**导入**（Excel/模板）批量建学生；导入时若含家长信息，**自动同步到家长联系**。
- 学生详情展示座次与职务，便于课堂快速定位。

### 5.3 考试管理
- 新建考试：默认勾选语文/数学/英语/科学/品德，其他科目默认不勾选（不参与考试）。
- 每次考试可记录各科成绩，自动排名（显示全班）。

### 5.4 成绩管理
- 录入/导入成绩，按考试查看排名、均分、优秀率、及格率、分数段。
- 「考试一键分析」可调用 AI 自动生成「本次考试分析情况描述」，并回写到考试记录。

### 5.5 座位表
- 可视化网格排座，学生分布在讲台两侧；使用中座次框用醒目颜色区分。
- 支持按「第 X 行 · 第 Y 列」编码，与新增/导入学生的座位号编码一致。

### 5.6 随机分组
- 按人数、按列等多种方式随机分组，课堂活动快速分队。

## 六、AI 智能生成（备课提效）

> 需管理员在后端「设置」页配置 AI 密钥与模型；未配置时页面会提示「未配置 AI 密钥」，不会崩溃。

- **知识点生成**：填年级/科目/教材/学期 + 主题 → 生成结构化讲解（定义/重难点/易错点/例题/复习建议）→ 可存入知识库。
- **优选试卷生成**：填年级/科目/年份/主题 → 生成贴近真实考纲的试卷（题目+答案+解析，模拟题明确标注）→ 存入试卷库。
- **考试一键分析**：选考试 → 前端汇总成绩统计 → AI 生成分析描述 → 回写考试记录。

## 七、语文 / 数学 / 英语专项

### 语文专项（7）
古诗词助手、汉字听写、阅读理解生成、小作文助手、成语词典、拼音标注、作文素材——输入主题/年级，AI 即时生成可复制的练习或讲解内容。

### 数学专项（6，本地算法）
- 口算生成：一键出加减乘除口算题。
- 竖式计算：输入算式，展示标准竖式过程与结果。
- 口算答题卡：生成答题卡并自动批改。
- 乘法口诀：口诀表与抽查练习。
- 单位换算：长度/面积/体积/质量/时间等常用单位互转。
- 错题本：本地拍照/文字记录错题，按学科分类，离线可查。

### 英语专项（8）
单词卡片、句型练习、英语听力、语法练习、情景对话、单词拼写、口语练习、英语爽文——AI 生成对应练习素材，可复制到备课文档。

## 十一、数据管理与教师行政

- **教学备课类**：课表、考勤、作业、公告、资源、教案、教案模板、知识点、优选试卷、试卷查询、听课记录——统一 CRUD 录入与查询。
- **班级事务类**：轮值表、值日配置、班费、班级活动、待办、笔记、抽签历史。
- **学生发展类**：成长记录、行为记录、获奖记录、奖项类别、加减分、积分记录、小组评分、家长联系、通知模板。
- **教师行政类**：教师通讯录、工作日志。

> 以上数据均保存在云端数据库，按教师隔离；换手机登录同一微信账号数据不丢。

---

## 十二、数据存储与隐私

- **存在哪里**：班级、学生、成绩等业务数据存于后端云数据库；登录态（token）仅存于你本机小程序缓存。
- **密钥安全**：微信 AppSecret、AI 密钥**只在后端「设置」页配置**，手机端拿不到，降低泄露风险。
- **数据隔离**：系统用你的教师身份过滤数据，看不到其他老师的班级内容。
- **本地工具数据**：错题本、文案模板库保存在你本机（uni.storage），不上云、不跨设备；如需备份请手动导出或截图。

## 十三、常见问题（FAQ）

**Q1：打开小程序提示「网络错误 / 请求失败」？**
A：通常是后端地址未配置或域名未备案。请联系管理员确认 `API_BASE` 与微信公众平台「服务器域名」已正确设置（详见部署手册）。

**Q2：AI 工具点生成没反应 / 提示未配置？**
A：后端「设置」页还没填 AI 密钥或模型地址。请管理员配置后重试。

**Q3：为什么有些工具不用联网也能用？**
A：18 款小游戏、口算、倒计时、计算器、单位换算、乘法口诀、错题本、文案模板库是本地逻辑，专为课堂弱网/无网场景设计。

**Q4：我在 A 手机录入的数据，B 手机会有吗？**
A：联网录入的班级/学生/成绩等数据会同步（云端）；但错题本、文案模板库是本机存储，不跨设备。

**Q5：数据会丢吗？**
A：云端数据由腾讯云数据库自动备份（管理员开启）；本机数据（错题本/模板）需自行注意，换手机前建议导出。
