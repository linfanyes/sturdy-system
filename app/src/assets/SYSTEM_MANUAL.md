## 🚀 快速开始

1. 双击 `start.bat`（Windows）
2. 浏览器自动打开 **http://localhost:5201**
3. 输入称呼 + 登录码 **`1314520`** 登录
4. 首次登录弹出"首次设置"，选择**任教学期**（如 2026 春季学期）和**任教学科**（多选）
5. 停止服务：双击 `stop.bat`

> 所有数据保存在浏览器 localStorage，**不联网**也可正常使用；AI 对话需联网 + API Key。

## 📚 功能模块

### 🏠 工作台
- 学期切换器（右上角 📅 图标）
- 今日待办 / 今日课表 / 今日公告 / 今日截止作业 等聚合信息
- 顶部消息铃铛汇总当天关键事项，一键跳转

### 🏫 班级管理
- 增删班级、设置班主任、班级口号、主题色
- **学期设置**：选择本学期涉及学科（系统会按此过滤其他模块）
- 班级整体同步：把某班级及学生整体同步到另一个班级（可保留或删除源班级）
- **班级职务设置**：在左侧班级卡片编辑按钮下方进入，可为每个班级勾选/自定义班长、副班长、学习委员、各学科课代表、组长等职务，并分配学生；保存后学生档案中的职务会同步更新
- **课表导入**：支持 TXT/Excel 模板 + **AI 智能识别**（任意格式）

### 🧒 学生管理
- 增删改学生（单条 / 批量）
- 学生档案：基本信息、家长联系方式、座位号、标签、备注
- **班级职务**：学生表单中的职务改为下拉框，选项来自「班级职务设置」中配置的职务
- **批量导入**：TXT/Excel 模板 + **AI 智能识别**（不用按模板）

### 📅 我的课表
- 1 张表展示，每日每课显示班级+任课科目
- 同一时间多个班级课程**堆叠**显示
- 双击课表单元格可直接修改科目和执教老师
- 打印时只显示课表表格和「用户名 + 的课表」表头
- 课表排版：编辑以**草稿**暂存，点击"保存到课表"才生效；排版后打印也只显示课表表格和「班级名称 + 课程表」表头
- 切换班级前若有未保存草稿会提示确认
- 午自习 / 课后服务**默认显示班主任**

### 📋 考试管理
- 考试时间、关联班级、关联科目、各科满分
- 支持语数英默认 100 分、其他科目默认 50 分
- 与成绩管理联动，先建考试再录成绩

### 📊 成绩管理
- 新增考试：语数英默认满分 100，其他默认 50
- 录入：单科 / 多科批量
- **综合分析弹框**（原左侧顶部「综合分析」菜单已合并到此）：
  - 6 项综合指标卡（参考人次 / 综合平均 / 最高 / 最低 / 及格率 / 优秀率）
  - **班级分数段分布**（10 段柱状图，含及格线 60 / 优秀线 90）
  - 各科平均分对比柱状图
  - 各科及格率 / 优秀率横向条形
  - 各科分数段分布对比表
  - 各科详情卡片 + 学科内迷你柱状图
  - 学生综合排名（前 5 / 后 5）
  - **成绩趋势**页签：查看指定学生多次考试的成绩变化曲线
- **批量导入**：TXT/Excel 模板 + **AI 智能识别**（支持图片识别录分）

### ✅ 考勤管理
- 每日打卡：出勤 / 迟到 / 旷课 / 请假
- 按班级、按日期快速登记与统计
- 工作台消息中心自动汇总今日考勤异常

### 📝 作业登记
- 布置 / 登记 / 状态追踪
- 状态：未布置 / 已布置 / 已收 / 已批改 / 待批改
- 支持开始日期与截止日期，今日截止作业会出现在工作台

### 📣 班级公告
- 置顶 / 一键结束 / 结束时间展示
- 可折叠已结束公告
- 支持班级维度公告与全校公告

### 📞 家长联系
- 记录学生家长联系方式与沟通记录
- 方便班主任留存家校沟通痕迹

### 🌱 成长档案
- 按学生记录成长点滴、获奖、作品、评语等
- 长期追踪学生发展轨迹

### 👀 行为观察
- 记录学生课堂/课间的行为表现
- 支持正向与待改进行为，辅助家校沟通

### 💰 班费管理
- 班费收入、支出、结余记录
- 明细清晰，支持按班级管理

### 🎉 班级活动
- 记录班级活动策划、参与人员、活动总结
- 留存班级集体记忆

### 📋 轮值表
- 安排班级每日值日生、值日任务
- 支持周期排班与快速查看

### 📓 工作日志
- 教师每日工作记录与反思
- 支持分类、搜索、回顾

### 📇 教师通讯录
- 同事信息 + 任教班级/学科分配
- 支持收藏、搜索、备注

### 📚 教学资源
- 教案、课件、网址等教学资源管理
- 支持分类、标签、链接跳转

### 📒 我的笔记
- 标题 + 内容 + 分类
- 支持置顶、收藏、搜索

### ✅ 待办
- 标题 + 日期 + 完成状态
- 今日待办在工作台聚合提醒

### 🏅 奖惩
- 学生 + 类型（表扬 / 批评）+ 分数 + 原因 + 日期
- 与学生档案联动，支持按班级筛选

## 🤖 AI 对话

> 路径：右上角 🤖 按钮 / 工具箱 → AI 对话

### 核心能力
- **自动注入系统数据**：班级、学生、成绩、课表、笔记、待办、奖惩等全部可用
- **自动注入本说明书**：AI 知道系统怎么用，回答更精准
- **多模态附件**：
  - 文本/代码（.txt .md .csv .json ...）自动读取全文
  - 图片（.png .jpg）走多模态识别
- **多会话**：左侧可建/删多个对话
- **回复操作**：复制 / 添加到笔记 / 重新生成

### 配置 API Key
1. 进入 AI 对话 → 右上角「设置」
2. 填入：
   - **Base URL**（默认 `https://dashscope.aliyuncs.com/compatible-mode/v1`）
   - **API Key**（如 `sk-...`）
   - **模型**（默认 `qwen3.7-plus`，可选 `qwen3-vl-plus`，可自定义）
3. 点保存

> Key 仅保存在本地浏览器，**不会上传**到任何服务器。

### 提问示例
- "帮我总结一下三年一班期中考试情况"
- "李明的数学最近一次考试多少分？"

- "这次考试及格率最高的科目是哪个？"
- "这个系统怎么修改登录码？"

## ✨ AI 智能识别导入

所有批量导入（课表 / 学生 / 成绩）都支持 **AI 智能识别模式**：

- **不按模板也能导入**：把任意格式的文本粘进去，AI 自动解析
- **流式进度**：实时显示 AI 解析状态
- **解析后预览**：自动切换到手动模式让你确认无误再保存

使用方法：
1. 打开"批量导入"弹框
2. 切换到 "✨ AI 智能识别" Tab
3. 把任意格式的文本粘进文本框
4. 点 "AI 解析"
5. AI 解析完后自动切回手动 Tab，可继续编辑/确认/保存

## 🧪 测试数据生成器

**入口**：登录后（登录码 `1314520`）在地址栏访问 `#/dev/seed`（即 `http://localhost:5201/#/dev/seed`）。

**操作步骤**：
1. 进入页面后，点击「一键生成测试数据」按钮。
2. 工具会先**清空当前账号已有的**班级 / 学生 / 考试 / 成绩 / 课表 / 笔记 / 待办 / 作业 / 教师 / 公告 / 教学资源等数据，再生成全套新数据（避免重复叠加）。

**生成内容**：
- 2 个班级（三年级一班 / 二班，班主任均为当前登录称呼）
- 30 名学生（每班 15 人）
- 自动排课（周一至周五，每天 8 节）
- 4 次考试（一班 3 次：第一单元 / 期中 / 第二单元；二班 1 次：期中）
- 各科成绩（按学生姓名生成的稳定真实分数，含少量缺考/满分）
- 8 条奖惩记录
- 5 条作业（含今日截止，方便在工作台查看）
- 6 条待办（含 1 条已完成）
- 4 篇笔记（含 1 置顶 + 1 收藏）
- 11 位教师信息
- 5 条班级 / 全校公告
- 5 个教学资源链接

**清空数据**：
- 在测试数据生成器页面点击「清空全部」，可清除当前账号所有上述数据（含教学资源、公告、作业、教师、笔记、待办），此操作不可撤销。

**数据存储**：全部写入当前登录用户隔离的 `localStorage`（`trace.{userId}.*`）。如需在自己常用的浏览器中查看，请先登录对应账号再运行生成，或在生成所用的同一浏览器中访问工作台。

## 🛠️ 常见问题

**Q: 启动报「未检测到 Node.js」？**
A: 请先安装 Node.js 18+ ：https://nodejs.org/zh-cn/download/

**Q: 端口 5201 被占用？**
A: 使用环境变量 `PORT=xxxx` 指定其他端口。

**Q: 双击 start.bat 没反应？**
A: 常见原因：Node.js 未安装 / 端口 5201 被占用 / 系统编码异常。脚本会自动尝试清理端口冲突。

**Q: 浏览器打开是空白？**
A: 查看 `app.log` 是否有错误。常见原因：Node 版本过低 / 端口冲突 / 依赖未装完。

**Q: AI 提示「请先配置 API Key」？**
A: 打开 AI 对话 → 右上角「设置」→ 填入 API Key。Key 仅保存在本地浏览器，不会上传。

**Q: 不同老师的数据会串吗？**
A: 不会。本系统按登录人**完全隔离**所有数据。

**Q: 如何切换任教学期？**
A: 工作台右上角 📅 图标 → 选择学期 → 自动保存。

**Q: 如何重置 AI 设置？**
A: AI 对话 → 设置 → 右下角"恢复默认"。

**Q: 如何快速体验全部功能？**
A: 开发模式下登录后访问 `#/dev/seed`，一键生成测试数据。

## 🛡️ 数据隔离机制

- 系统为每位老师生成独立的 `userId`
- 所有业务数据使用 `trace.{userId}.{base}` 形式命名
- 登出后该老师数据仍保留；下次再登录自动恢复
- 支持多账号轮换：李老师登录 → 登出 → 王老师登录，**两人数据完全独立**

| 模块         | localStorage key                       |
| ------------ | -------------------------------------- |
| 班级 / 学生  | `trace.{userId}.classes`               |
| 课表 / 作业 / 公告 / 资源 / 考勤 | `trace.{userId}.school`  |
| 考试计划     | `trace.{userId}.exams`                 |
| 成绩         | `trace.{userId}.grades`                |
| 笔记         | `trace.{userId}.notes`                 |
| 待办         | `trace.{userId}.todos`                 |
| 奖惩         | `trace.{userId}.rewards`               |
| 同事 / 通讯录 | `trace.{userId}.teachers`             |
| AI 配置      | `trace.{userId}.ai-settings`           |
| AI 对话      | `trace.{userId}.ai-chats`              |
